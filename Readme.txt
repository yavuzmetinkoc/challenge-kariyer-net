Bu çalışma iş başvurusu yapmış olduğum Kariyer.Net şirketinin göndermiş olduğu çalışma sonucunda yapılmıştır.

Programda Vue.js kullanılarak geliştirilmiş olup gelen json verideki ilanlar üzerinde arama yapmayı amaçlamaktadır.

Program VsCode ortamında geliştirilmiş olup Live Server eklentisi kullanılarak test edilmiştir.

Programda Json formatında veri okumak için sunucum üzerinden veri çekilmektedir. Bu veri çekme işlemi sırasında uzantı kullanılması gerekmektedir.
Chrome kullananlar için Allow Control Allow Origin uzantısını kullanarak verilerin görünmesini sağlayabilirsiniz.
